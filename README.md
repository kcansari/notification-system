# Notification System

A simple project to learn the Factory Method pattern by building a notification system that sends messages through different channels.

## Why This Project?

Learning design patterns through real-world examples! This project shows how the Factory Method pattern works by sending notifications through Email, SMS, Slack, and Discord.

**Note:** This is an educational project, not production-ready. Built to practice and understand the Factory Method design pattern.

## Get Started

```bash
pnpm install && pnpm dev
```
